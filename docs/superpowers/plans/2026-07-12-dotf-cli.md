# dotf CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A `dotf` veneer CLI (apply/doctor/update/test) installed as a stow package, plus a curl-able `bootstrap.sh` that takes a fresh Ubuntu server to a converged machine.

**Architecture:** Two new bash files. `bootstrap.sh` (repo root) is the curl target: Ubuntu gate → git install → verified HTTPS clone to `~/.dotfiles` → `setup.sh`. `home/dotf/.local/bin/dotf` is a single-file CLI stowed into `~/.local/bin`; it discovers the repo root by resolving its own symlink and delegates to the existing scripts. Existing `setup.sh`/`apply.sh`/`doctor.sh` logic is untouched (one read-only check is added to `doctor.sh`).

**Tech Stack:** bash, GNU Stow, git; static verification via `tests/ubuntu-config.bash` (the guard), `bash -n`, `zsh -n`.

**Spec:** `docs/superpowers/specs/2026-07-12-dotf-cli-design.md` — read it before starting.

**Target release:** v2.0.0 (current release: v1.0.0). Tagging/releasing happens once at the end — see "Release" after the final task; individual tasks never tag.

## Global Constraints

- **NEVER run `git commit`.** The user commits manually with `/pc:commit`. Tasks end by staging (`git add`) only. No AI co-author trailers ever.
- Ubuntu-only, headless: no macOS artifacts, no brew/cask, no desktop tooling (guard-enforced).
- No secrets or auth state: bootstrap clones via HTTPS `https://github.com/VimukthiShohan/ubuntu-server-dotfiles.git` — never `git@`/`ssh://`, never tokens.
- Canonical clone dir on servers: `~/.dotfiles` (override env var: `DOTFILES_DIR`).
- All scripts: `set -euo pipefail`, idempotent, safe to re-run. `doctor.sh` stays strictly read-only.
- `dotf` is a **veneer**: it delegates to existing scripts and never grows its own converge logic (see `CONTEXT.md`).
- This plan is executed on the macOS authoring machine: verification is static checks plus in-place smoke runs. Never run `./apply.sh`, `./setup.sh`, or `stow` here.

- [ ] **Step 3: Write bootstrap.sh**

Create `bootstrap.sh` at the repo root with exactly this content:

```bash
#!/usr/bin/env bash
# bootstrap.sh - fresh Ubuntu server -> converged machine in one step.
# Usage: curl -fsSL https://raw.githubusercontent.com/VimukthiShohan/ubuntu-server-dotfiles/main/bootstrap.sh | bash
# Precondition: run as a non-root user with sudo access.
#
# Whole script is functions with a trailing `main "$@"` so nothing executes
# until bash has parsed the full file (required for `curl | bash`), and main
# detaches stdin so no inner command can eat streamed script text.

set -euo pipefail
trap 'echo "!! bootstrap.sh: step above failed. Fix it, then re-run this script."' ERR

REPO_HTTPS="https://github.com/VimukthiShohan/ubuntu-server-dotfiles.git"
TARGET="${DOTFILES_DIR:-$HOME/.dotfiles}"

verify_existing_clone() {
  local origin
  if ! origin="$(git -C "$TARGET" remote get-url origin 2>/dev/null)"; then
    echo "!! $TARGET exists but is not a git clone with an 'origin' remote." >&2
    echo "   Move it aside and re-run bootstrap." >&2
    exit 1
  fi
  case "$origin" in
    *VimukthiShohan/ubuntu-server-dotfiles*) ;;
    *)
      echo "!! $TARGET exists but its origin is '$origin', not ubuntu-server-dotfiles." >&2
      echo "   Move it aside (or set DOTFILES_DIR elsewhere) and re-run bootstrap." >&2
      exit 1
      ;;
  esac
}

main() {
  exec </dev/null

  if [[ -r /etc/os-release ]]; then
    # shellcheck disable=SC1091
    . /etc/os-release
  fi
  if [[ "${ID:-}" != "ubuntu" ]]; then
    echo "!! bootstrap.sh targets Ubuntu. Detected ID='${ID:-unknown}'." >&2
    exit 1
  fi

  if [[ "$(id -u)" -eq 0 ]]; then
    echo "!! Run bootstrap as a non-root user with sudo, not as root." >&2
    exit 1
  fi

  if ! command -v git >/dev/null 2>&1; then
    echo "==> Installing git"
    sudo apt-get update
    sudo apt-get install -y git
  fi

  if [[ -e "$TARGET" ]]; then
    echo "==> $TARGET already exists; verifying it is this repo"
    verify_existing_clone
  else
    echo "==> Cloning dotfiles to $TARGET"
    git clone "$REPO_HTTPS" "$TARGET"
  fi

  echo "==> Running setup"
  "$TARGET/setup.sh"

  echo
  echo "==> Bootstrap complete. Start a new login shell (log out and back in);"
  echo "    'dotf' is then on PATH: dotf apply | doctor | update | test"
}

main "$@"
```

- [ ] **Step 3: Write the dotf script**

Create `home/dotf/.local/bin/dotf` with exactly this content:

```bash
#!/usr/bin/env bash
# dotf - veneer over this repo's scripts. Delegates only; owns no converge logic.
# Spec: docs/superpowers/specs/2026-07-12-dotf-cli-design.md

set -euo pipefail

usage() {
  cat <<'EOF'
Usage: dotf <command>

Commands:
  apply [--fresh]   Converge this machine (apply.sh; --fresh adopts pre-existing files)
  doctor            Read-only drift check (doctor.sh; exit 1 on drift)
  update            git pull --ff-only in the repo, then converge
  test              Guard test + bash -n + zsh -n (pre-commit verification)
  help              Show this help
EOF
}

repo_root() {
  local self root
  self="$(readlink -f "${BASH_SOURCE[0]}")"
  # <root>/home/dotf/.local/bin/dotf -> strip dotf, bin, .local, dotf, home
  root="$(dirname "$(dirname "$(dirname "$(dirname "$(dirname "$self")")")")")"
  if [[ ! -f "$root/apply.sh" ]]; then
    echo "!! dotf: cannot locate repo root (resolved: $self, derived: $root)" >&2
    exit 1
  fi
  printf '%s\n' "$root"
}

cmd_update() {
  local root="$1"
  if [[ -n "$(git -C "$root" status --porcelain)" ]]; then
    echo "!! dotf update: working tree at $root is dirty; commit or stash first." >&2
    exit 1
  fi
  git -C "$root" pull --ff-only
  exec "$root/apply.sh"
}

cmd_test() {
  local root="$1"
  bash "$root/tests/ubuntu-config.bash"
  bash -n \
    "$root/setup.sh" \
    "$root/apply.sh" \
    "$root/doctor.sh" \
    "$root/bootstrap.sh" \
    "$root/setup/install-tools.sh" \
    "$root/setup/tools/installers.sh" \
    "$root/tests/ubuntu-config.bash" \
    "$root/home/dotf/.local/bin/dotf"
  zsh -n "$root/home/zsh/.zshrc" "$root"/home/zsh/.config/zsh/*.zsh
  echo "dotf test: all checks passed"
}

main() {
  local cmd="${1:-help}"
  if (( $# > 0 )); then shift; fi

  case "$cmd" in
    apply)          exec "$(repo_root)/apply.sh" "$@" ;;
    doctor)         exec "$(repo_root)/doctor.sh" ;;
    update)         cmd_update "$(repo_root)" ;;
    test)           cmd_test "$(repo_root)" ;;
    help|-h|--help) usage ;;
    *)              usage >&2; exit 1 ;;
  esac
}

main "$@"
```
